          Implementation notes
          

            
• AES‑GCM (WebCrypto) with PBKDF2 key derivation (random 16‑byte salt, 12‑byte IV).
• LSB embeds a header magic PXPF, payload length, salt, IV, then ciphertext.
• Prompts (randomized): tilt left→right, rotate ~360°, cover→uncover lens (brightness change).
• Metadata: device/userAgent, ts, resolution, randomized nonce, (optional) GPS, motion stats, prompt outcomes.

            
• Works best over HTTPS + mobile browsers. iOS requires user gesture for motion permission.
• This is a demo — not production‑grade security. Use robust threat models, anti‑ml attacks, watermarking, and server‑side checks.
